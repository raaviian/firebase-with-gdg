# Build it with Firebase

## Tech Stack

**Client:** TailwindCSS

**Server:** Node, Express

## Installation

Install Firebase CLI with npm

```bash
  npm install -g firebase-tools
  firebase --version
  firebase login
  firebase init
```

Install Node.js with npm

    https://nodejs.org/en
